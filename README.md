# Interactive Birthday Card

A small HTML/CSS/JS project demonstrating an interactive birthday card with animations.

## Features

- Floating sealed envelope on the homepage
- Envelope opens with animation when clicked
- A 3D birthday card slides out and can be rotated 360° on the X-axis via drag
- Smooth animations using GSAP and CSS keyframes
- Responsive design suitable for GitHub Pages hosting

## Files

- `index.html` - markup for the envelope and card
- `style.css` - styles, animations, and layout
- `script.js` - interactivity and drag rotation logic

## Deployment

1. Push the repository to GitHub.
2. In repository settings, enable **GitHub Pages** and set the source to `main` branch (or whichever branch you're using).
3. Navigate to the provided URL; the animations work in most modern browsers.

Enjoy customizing the design for your own special occasions! 😊
